# Tour

A simple website to give a few details of the Continets, Oceans and a few facts of the world.

## Installation

Follow these stepss to set up the prject locally;

1. Check if your have the following software installed;
   -Node.js
   -npm
   -Git

2. Clone the project;

git clone https://erickmasila-sudo.github.io/Tour.git

## Table of Contents

1. index.html
  -This is the file for the homepage.
  -It contains 4 buttons to help navigate through the other webapges at the top of the page

2. Continents.html
   -This is the file that contanis infomartion of the 7 continets
   -It has the navigation Class in its header which is used to move between the other files.
   -The Contnients information come in the form of cards which use the hover effect to allow for content to only be displayed when one hovers over it
       The Hover effect:
         In .Holder .Element{
            p{display: none;}
            a{display: none;}
         }
         In .Holder .Element:hover{
            p{display: content;}
            a{display: contents;}
         }

3. Oceans. html

         }

4. TheWorldMap.html
